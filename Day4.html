Semantic HTML

This part of tutorial shows only dividing tags and how to use them in already prepared code. I am going to fix code from day 3 then and list all new tags in this paragraph.

  • <header> container for navigational links and introductory content;
  • <nav> is used to define a block of navigation links such as menus and tables of contents;
  • <main> is used to encapsulate the dominant content within a webpage;
  • <footer> is the content at the bottom of the subject information;
  • <section> seperates elements of text within same theme;
  • <article> contain a block of text that makes senese on its own;
  • <aside> used for additional informatin that is not requred to understand article;
  • <figure> tag for media such as images, videos, music, etc.;
  • <figcaption> allows to add figure description;
  • <audio> and <video> for accordingly sound and image;
  • <embed> tag, which can embed any media content including videos, audio files, and gifs from an external source.
